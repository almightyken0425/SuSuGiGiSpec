# 主題設定: ThemeSettingsScreen

## 畫面目標

- 提供瀏覽並切換 App 配色主題的介面

---

## 觸發情境

- 以 Modal 呈現
- 由偏好設定的主題入口觸發

---

## 線框圖

```text
+--------------------------------+
| X      Themes         Done     |
+--------------------------------+
| [ Preview ]   [ Preview ]      |
| Name          Name             |
| [ Selected ]                   |
|                                |
| [ Preview ]   [ Preview ]      |
| Name          Name             |
+--------------------------------+
```

---

## 佈局

### 導覽列

- 關閉按鈕
- 配色主題 標題
- 完成按鈕

### 主題列表

- 主題卡片
  - 配色預覽區
  - 主題名稱
  - **IF** 當前已選主題:
    - 顯示選取標記

---

## 互動

- **點按關閉按鈕:**
  - 關閉 Modal，不套用變更

- **點按主題卡片:**
  - 更新本機選取狀態
  - 該卡片顯示選取標記

- **點按完成按鈕:**
  - 呼叫 switchTheme，套用選取主題
  - 關閉 Modal

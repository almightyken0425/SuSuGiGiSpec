# 商業模式與定價策略

## 核心戰略

本文件依據產品定義的四大產品個體，制定相應的收費策略與商業模式。

- **混合式多層級模型:**
    - 結合 **工具軟體 SaaS** 與 **數據服務 DaaS** 的混合模式
    - C 端透過工具價值獲取海量用戶與微觀數據
    - B 端透過數據價值變現宏觀趨勢並回饋 C 端研發

---

## 1. 記帳 App + Firestore, Accounting App

**策略:** Freemium, 免費增值
**目標:** 獲取, Acquisition & 留存, Retention

- **Free Tier, 免費版**
    - **定價:** $0 / 月
    - **功能:**
        - 完整記帳功能, CRUD
        - 本地資料庫, WatermelonDB
        - 基礎報表, 圓餅圖、趨勢圖
        - Widget 小工具
    - **限制:** 無雲端同步、單一裝置、無進階報表

- **Standard Tier, 標準訂閱**
    - **定價:** $30 / 月, 或 $300 / 年
    - **價值主張:** 資料安全與跨裝置便利
    - **功能:**
        - **包含 Free Tier 所有功能**
        - **雲端同步:** 多裝置即時同步, iPad + iPhone
        - **自動備份:** 每日雲端備份
        - **共用帳本:** 支援雙人協作, 未來規劃

---

## 2. Web 複雜報表, Web Console

**策略:** Value-Based Pricing, 價值導向定價
**目標:** 提升 ARPU, Average Revenue Per User

- **Pro Tier, 專業訂閱**
    - **定價:** $90 / 月, 或 $900 / 年
    - **價值主張:** 駕馭複雜性，個人企業 ERP
    - **功能:**
        - **包含 Standard Tier 所有功能**
        - **Web 控制台存取權**
        - **進階篩選:** JQL 查詢介面
        - **客製化報表:** 自訂維度分析
        - **批量編輯:** 大量資料重整工具
        - **匯出:** 完整 CSV/Excel 匯出

---

## 3. AI 個人化建議, AI Advisor

**策略:** Add-on, 加值服務 或包含於頂級訂閱
**目標:** 增加黏著度與提供高階價值

- **AI Add-on, AI 顧問包**
    - **定價:** +$50 / 月, 需搭配 Standard 或 Pro
    - **價值主張:** 專屬財務顧問，預知未來
    - **功能:**
        - **異常偵測通知**
        - **現金流預測模擬**
        - **自然語言財務問答**
        - **財務健康診斷報告**

*註: 初期可考慮將 AI 功能直接包含於 Pro Tier 以增加吸引力，待成熟後再獨立拆分*

---

## 4. 總經分析報表與 API, Macro Data Service

**策略:** B2B 訂閱與流量計費
**目標:** 數據變現, Data Monetization

- **Enterprise Data Service, 企業數據服務**
    - **定價:** 依專案報價 或 企業訂閱制, $500+ / 月
    - **客戶:** 市場研究機構、品牌商、金融機構
    - **產品:**
        - **市場情資儀表板:** 存取特定受眾的聚合消費數據
        - **總經數據 API:** 依 API 呼叫次數計費, Usage-Based
    - **合規:** 僅提供 K-Anonymity 聚合數據，絕無 PII

---

## 飛輪效應

1.  **免費版** 吸引大量用戶，建立數據基底。
2.  **標準版** 提供安全與便利，轉化付費用戶，支撐基礎營運。
3.  **專業版 & AI** 服務高價值用戶，提供深度洞察，提升獲利能力。
4.  **B 端數據** 變現宏觀價值，回饋 C 端研發，形成正向循環。

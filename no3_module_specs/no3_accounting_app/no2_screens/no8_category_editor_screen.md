# 類別編輯器規格: CategoryEditorScreen

## 畫面目標

- **提供:** 新增 或 編輯 類別 的統一介面
- **確保:** 所有自訂類別均映射到標準類別

```text
+--------------------------------+
| Close       Title       Save   |
+--------------------------------+
| Name: [ Input Text ]           |
| Icon: [ Icon Preview ] >       |
| Map To: [ Standard Cat ] >     |
|                                |
| Disable: [ Switch ]            |
|                                |
|           [Delete]             |
+--------------------------------+
```

## UI 佈局

- **呈現方式:** Modal 形式從底部彈出
- **頂部導航列:**
    - **內部元件:**
        - **關閉按鈕**
        - **標題**
            - **內容:** 依模式顯示 新增支出類別, 編輯收入類別
        - **儲存按鈕**
            - **可見性:** 僅所有必填欄位有效時才可點擊
- **核心欄位區:**
    - **類別名稱:**
        - **UI:** 文字輸入框, TextInput
        - **屬性:** 必填項
    - **圖示:**
        - **UI:** 顯示當前所選圖示的區域
        - **導航:** IconPickerScreen
        - **屬性:** 必填項
    - **標準類別映射:**
        - **UI:** 選擇器
        - **行為:** 點擊彈出 StandardCategory.json 列表
        - **邏輯:** 列表依 `categoryType` 過濾
        - **屬性:** 必填項
- **停用開關區:**
    - **UI:** Switch 元件
    - **標題:** 停用此類別
    - **可見性:** 僅 編輯模式 顯示
    - **邏輯:** 讀取 `disabledOn` 欄位狀態決定開關預設值
- **刪除按鈕區:**
    - **UI:** 紅色 刪除此類別 按鈕
    - **可見性:** 僅 編輯模式 顯示

## 核心邏輯

- **模式判斷:**
    - **觸發:** 畫面載入
    - **檢查:** 導航參數 `categoryId`
    - **IF** 有 `categoryId` (編輯模式):
        - **行為:** 從 本機 DB 讀取類別資料填入表單
    - **IF** 無 `categoryId` (新增模式):
        - **行為:** 表單為空白
        - **預設:** categoryType 預設為 支出
- **儲存邏輯:**
    - **觸發:** 點擊 儲存按鈕
    - **新增模式:**
        - **付費牆檢查:**
            - **檢查:**
                - `PremiumContext.isPremiumUser` 狀態
                - 本機 DB 類別數量
            - **IF** 免費版 AND 類別數量已達上限:
                - **導航:** PaywallScreen
            - **ELSE:**
                - **行為:** 於 本機 DB 建立新記錄
                - **欄位:** 必須設定 `updatedOn`
    - **編輯模式:**
        - **行為:**
            - 組合表單資料
            - **IF** 停用開關 開啟:
                - **欄位:** `disabledOn` 設為當前時間戳記
            - **IF** 停用開關 關閉:
                - **欄位:** `disabledOn` 設為 null
        - **儲存:** 更新 本機 DB 該筆記錄
        - **欄位:** 必須更新 `updatedOn`
    - **成功:**
        - **導航:** 關閉畫面, 返回 CategoryListScreen
- **刪除邏輯:**
    - **觸發:** 點擊 刪除按鈕
    - **行為:** 顯示確認對話框
    - **成功:**
        - **行為:** 軟刪除 本機 DB 該筆 `Category`
        - **欄位:** 必須設定 `deletedOn` 並更新 `updatedOn`
        - **導航:** 關閉畫面, 返回 CategoryListScreen

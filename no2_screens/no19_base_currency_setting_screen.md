# 主要貨幣設定: BaseCurrencySettingScreen

## 畫面目標

- 提供選擇 App 主要貨幣的介面

---

## 線框圖

```text
+--------------------------------+
| X     Base Currency      Done  |
+--------------------------------+
| [ Search Currency... ]         |
+--------------------------------+
| USD - United States Dollar  ✓  |
| TWD - New Taiwan Dollar        |
| JPY - Japanese Yen             |
| ...                            |
+--------------------------------+
```

---

## 佈局

### 導覽列

- 關閉 按鈕
- 主要貨幣 標題
- 完成 按鈕

### 搜尋列

- 貨幣關鍵字輸入框

### 貨幣列表

- 目前選取貨幣置頂
- 貨幣項目
  - `alphabeticCode`
  - `name`
  - **IF** 為目前選取貨幣:
    - 顯示選取標記

---

## 互動

- **點按關閉按鈕:**
  - 返回上一頁

- **輸入搜尋文字:**
  - 依貨幣代碼或名稱即時篩選列表

- **點按貨幣項目:**
  - 將此項目標為目前選取貨幣

- **點按完成按鈕:**
  - 呼叫 setBaseCurrency
  - 返回上一頁

# 定期交易設定: RecurringSettingScreen

## 畫面目標

- 提供設定交易或轉帳重複規則的介面

---

## 觸發情境

- 由 TransactionEditorScreen 或 TransferEditorScreen 以 Modal 呈現

---

## 線框圖

```text
+--------------------------------+
| Cancel      Repeat      Done   |
+--------------------------------+
|                                |
| Frequency: [ Monthly ] >       |
| Every: [ 1 ] Month(s)          |
|                                |
| End: [ Never ] >               |
+--------------------------------+
```

---

## 佈局

### 導覽列

- 取消按鈕
- 標題
- 完成按鈕

### 設定區

- 頻率選擇器
  - 每日
  - 每週
  - 每月
  - 每年
- 間隔輸入框
- 結束條件選擇器
  - 永不
  - 於特定日期
- **IF** 結束條件為特定日期:
  - 結束日期選擇器

---

## 互動

- **點按取消按鈕:**
  - 關閉 Modal

- **點按完成按鈕:**
  - 關閉 Modal

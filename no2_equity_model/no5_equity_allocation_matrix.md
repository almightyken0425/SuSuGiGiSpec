# 股權分配總表

## 核心目標

- 本文件是股權計算的最終總結
- 目的在於將產品組件的任務分配給角色定義的角色
- 藉此計算出每個角色在總理論價值中所佔的價值點數比例
- 這個比例是未來新資分配與股權結構的依據

## 角色價值分配, 施工價值

- **說明:** 依據產品組件的詳細估算彙整。
- **總施工價值:** 450,000 點

| 角色 | 負責範疇 | 估算價值點數 | 佔總價值比例 |
| :--- | :--- | :--- | :--- |
| **Mobile Dev** | App 核心、同步整合、報表 UI | 115,000 | 25.6% |
| **Backend Dev** | Sync Protocol、AI 整合、API Gateway | 70,000 | 15.6% |
| **Frontend Dev** | Web Console、JQL、B 端儀表板 | 120,000 | 26.7% |
| **UI/UX Design** | App/Web 視覺設計、互動體驗 | 35,000 | 7.8% |
| **AI Engineer** | 異常偵測、預測模型 | 70,000 | 15.6% |
| **Data Engineer** | ETL Pipeline、去識別化 | 40,000 | 8.9% |
| **總計** | | **450,000** | **100.0%** |

## 創辦人股權分配, 範例

- **說明:** 依據每個創辦人所承諾擔任的角色, 來計算其應得的股權。
- **情境:** 假設團隊有 3 位創辦人

| 創辦人 | 擔任角色 | 承諾角色總價值 | 最終股權比例 |

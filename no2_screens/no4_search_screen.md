# 搜尋: SearchScreen

## 畫面目標

- 提供搜尋所有交易與轉帳備註的介面，並作為導航至紀錄編輯畫面的入口

---

## 觸發情境

- 以 Modal 呈現
- 由點擊首頁搜尋入口觸發

---

## 線框圖

```text
+--------------------------------+
| X    Search Note...            |
+--------------------------------+
| [Icon] Category Name           |
|        Note Text               |
|         Amount      Date       |
|                                |
| [Icon] Transfer                |
|        Note Text               |
|         Amount      Date       |
+--------------------------------+
```

---

## 佈局

### 導覽列

- 關閉按鈕
- 搜尋輸入框
  - 進入時自動聚焦
  - 搜尋圖示
  - 清除按鈕

### 搜尋結果區

- **IF** 搜尋框為空:
  - 顯示輸入關鍵字以搜尋交易備註提示
- **IF** 無符合結果:
  - 顯示找不到符合關鍵字的紀錄提示
- 搜尋結果列表
  - 交易與轉帳各自最多顯示 50 筆
  - 依日期由新至舊排序
- 搜尋結果列表項目
  - 圖示
  - 類別名稱或轉帳標籤
  - 備註
    - 關鍵字高亮顯示
  - 金額
  - 日期

---

## 互動

- **點按關閉按鈕:**
  - 關閉 Modal

- **輸入搜尋文字:**
  - 延遲 300 毫秒後自動觸發搜尋，更新搜尋結果列表

- **點按搜尋結果列表項目:**
  - **IF** 交易紀錄:
    - 導航至 TransactionEditorScreen
  - **IF** 轉帳紀錄:
    - 導航至 TransferEditorScreen

# MVP 範圍定義

## MVP 核心架構

- [ ] **本機儲存架構 Local-First:** App 核心 CRUD 功能必須能在 100% 離線狀態下於本機資料庫 Local DB 運作。
    
- [ ] **使用者認證:** 強制使用 Firebase Auth 的 Google 登入, 用於帳號綁定。
    
- [ ] **付費牆 PaywallScreen:** 串接 App Store 或 Google Play 訂閱機制, 例如 RevenueCat。
    

## 免費版功能

- [ ] **核心記帳:**
    
    - [ ] 新增, 編輯, 刪除支出 Expense 交易。
        
    - [ ] 新增, 編輯, 刪除收入 Income 交易。
        
    - [ ] 新增, 編輯, 刪除同幣別轉帳 Transfer。
        
- [ ] **功能限制:**
    
    - [ ] **帳戶限制:** 最多 3 個帳戶。
        
    - [ ] **類別限制:** 最多 10 個自訂類別。
        
- [ ] **首頁 HomeScreen:**
    
    - [ ] 完整儀表板功能, 包含時間區間切換, 帳戶篩選器, 摘要與列表視圖。
        
- [ ] **帳戶/類別管理:**
    
    - [ ] 建立, 查看, 編輯, 刪除帳戶, 受限於 3 個。
        
    - [ ] 建立, 查看, 編輯, 刪除類別, 受限於 10 個。
        
- [ ] **首次啟動流程:**
    
    - [ ] 偵測新用戶, 並在本機資料庫建立預設幣別, 時區, 預設帳戶與類別。
        
- [ ] **設定:**
    
    - [ ] 帳戶管理入口。
        
    - [ ] 類別管理入口。
        
    - [ ] 偏好設定入口, 包含主要貨幣, 時區, 語系切換, 登出功能。
        
    - [ ] **國際化 i18n:** App 必須支援至少兩種語言, 例如 zh-TW 與 en。
        
- [ ] **搜尋功能:**
    
    - [ ] 支援搜尋 Note 欄位, 僅限本機資料。
        

## 付費版功能

- [ ] **付費功能 雲端同步與備份:**
    
    - [ ] 實作「**每日批次同步**」引擎。
        
    - [ ] 實作「**立即同步 Sync Now**」手動按鈕, 包含 5 分鐘冷卻機制。
        
- [ ] **付費功能 解鎖限制:**
    
    - [ ] **無限**帳戶。
        
    - [ ] **無限**類別。
        
- [ ] **付費功能 多幣別功能:**
    
    - [ ] 建立/管理外幣帳戶。
        
    - [ ] 支援**跨幣別**轉帳。
        
    - [ ] **匯率管理**介面。
        
- [ ] **付費功能 自動化:**
    
    - [ ] 定期交易 Schedules 的建立, 編輯, 刪除及自動產生。
        
- [ ] **付費功能 資料遷移:**
    
    - [ ] CSV 檔案匯入資料功能。

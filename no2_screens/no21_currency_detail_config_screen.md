# 貨幣顯示格式: CurrencyDetailConfigScreen

## 畫面目標

- 提供設定單一貨幣顯示格式的介面，含千分位模式與小數位數

---

## 線框圖

```text
+--------------------------------+
| X        USD             Done  |
+--------------------------------+
| United States Dollar           |
|                                |
| THOUSANDS UNIT MODE            |
| Omit last 3 zeros    [ Off ]   |
|                                |
| DECIMAL PLACES                 |
| 0                              |
| 1                              |
| 2                           ✓  |
| 3                              |
| Reset to Default (2)           |
+--------------------------------+
```

---

## 佈局

### 導覽列

- 關閉 按鈕
- 貨幣代碼 標題
- 完成 按鈕

### 貨幣名稱標題

- `name`

### 千分位模式 分組

- 分組標題
- 千分位切換列
  - 標題文字
  - 說明文字
  - 切換開關
  - **IF** 千分位開關為開啟:
    - 顯示啟用狀態的說明文字
  - **IF** 千分位開關為關閉:
    - 顯示標準顯示的說明文字

### 小數位數 分組

- 分組標題
- 小數位數選項：0、1、2、3
  - **IF** 為目前選取位數:
    - 顯示選取標記
- 重置為預設值 選項
  - 顯示該貨幣的預設位數
    - **IF** 為 TWD:
      - 預設位數為 0
    - **IF** 為其他貨幣:
      - 預設位數取自 `minorUnits`

---

## 互動

- **點按關閉按鈕:**
  - 返回上一頁

- **切換千分位開關:**
  - 更新目前千分位設定

- **點按小數位數選項:**
  - 將此選項標為目前選取位數

- **點按重置為預設值:**
  - 呼叫 resetCurrencyFormat
  - 將目前選取位數回復為該貨幣預設位數

- **點按完成按鈕:**
  - 呼叫 setCurrencyFormat
  - 返回上一頁

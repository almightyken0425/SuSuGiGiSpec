# 付費牆規格: PaywallScreen

## 畫面目標

- **傳達:** Premium 版本的雙重價值
    - 功能解鎖
    - 雲端同步
- **提供:** 無摩擦的介面引導使用者完成訂閱
- **處理:** 購買與恢復購買的邏輯

---

## 觸發情境

- **主動觸發:**
    - **來源:** 點擊 設定主頁 上的 升級至 Premium 按鈕
- **被動觸發:**
    - **來源:** 嘗試存取任何 付費功能 時
    - **備註:** 由各功能畫面自行檢查 `isPremiumUser` 狀態並呼叫此畫面

## 核心邏輯

- **串接服務:**
    - **要求:** 串接跨平台購買服務
    - **範例:** RevenueCat
- **動態價格載入:**
    - **觸發:** 畫面載入
    - **行為:** 呼叫 `RevenueCat.getOfferings`
    - **禁止:** 在 App 內寫死價格
    - **顯示:** 從 RevenueCat 取回本地化價格字串
- **購買邏輯:**
    - **觸發:** 使用者選擇方案並點擊 CTA
    - **行為:** 呼叫 `RevenueCat.purchasePackage`
    - **成功:**
        - **更新:** 本機狀態 `isPremiumUser` 為 True
        - **提示:** 升級成功
        - **導航:** 自動關閉 PaywallScreen
    - **失敗:**
        - **提示:** 購買失敗 或 已取消
- **恢復購買邏輯:**
    - **觸發:** 點擊 恢復購買 連結
    - **行為:** 呼叫 `RevenueCat.restorePurchases`
    - **成功:**
        - **更新:** 本機狀態 `isPremiumUser` 為 True
        - **提示:** 恢復成功
        - **導航:** 自動關閉 PaywallScreen
    - **失敗:**
        - **提示:** 找不到您的購買紀錄

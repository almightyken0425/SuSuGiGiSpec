# 匯率編輯器: CurrencyRateEditorScreen

## 畫面目標

- 提供新增特定幣別對匯率的介面

---

## 線框圖

```text
+--------------------------------+
| Cancel     Set Rate     Done   |
+--------------------------------+
|                                |
| [ From Cur ]      [ To Cur ]   |
|                                |
| [ Amount A ]  =  [ Amount B ]  |
|                                |
+--------------------------------+
```

---

## 佈局

### 導覽列

- 取消按鈕
- 設定匯率 標題
- 完成按鈕
  - **IF** 輸入欄位未填妥:
    - 不可點按

### 幣別對顯示區

- 來源幣別代碼
- 目標幣別代碼
- 不可修改

### 匯率輸入區

- 來源金額輸入框
- 等號標籤
- 目標金額輸入框

---

## 互動

- **點按取消按鈕:**
  - 返回上一頁

- **點按完成按鈕:**
  - 呼叫 createCurrencyRate
  - **IF** 操作成功:
    - 返回上一頁
  - **IF** 操作失敗:
    - 顯示錯誤提示

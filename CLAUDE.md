# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 專案概述

SuSuGiGi App 的產品規格文件庫，涵蓋從產品定位、股權模型、功能模組規格到開發管理政策的完整文件。無程式碼、無 build 流程。

---

## 目錄結構

- **no1_product_initiation** — 產品定位，包含 App 價值主張、產品哲學、產品定義、商業模式、命名討論
- **no2_equity_model** — 股權模型，定義貢獻者角色、積分換算、股權分配政策與操作管理
- **no3_module_specs** — 功能模組規格書
  - **no0_policy** — 跨模組政策，含 UI 設計規範、i18n 政策、錯誤處理、錯誤碼定義
  - **no1_interaction_flows** — 整體互動流程
  - **no2_user_management** — 使用者管理模組，含架構與邏輯
  - **no3_accounting_app** — 記帳模組，含架構、畫面、邏輯、資料模型、設計系統
- **no4_dev_management** — 開發管理
  - **no0_policy** — Git 工作流程、Coding Style、測試、Release 政策
  - **no1_mvp_planning** — MVP 規劃、檔案計畫、開發計畫
- **no5_marketing_strategy** — 地區市場推廣策略
- **no99_archive** — 已完成或封存的一次性文件

---

## 撰寫規範

所有規格文件依循 universal_writing_linter skill 的寫作政策，核心禁令包含禁止使用括弧、禁止數字編號列表、UI 元件須包裹反引號、標題不得使用數字排序。修改規格文件前請先閱讀 policy 目錄下的通用寫作政策。

# 商業模式與定價策略

## 核心戰略

- **混合式多層級模型:**
    - 結合 **工具軟體 SaaS** 與 **數據服務 DaaS** 的混合模式
    - C 端透過工具價值獲取海量用戶與微觀數據
    - B 端透過數據價值變現宏觀趨勢並回饋 C 端研發
- **價值階梯:**
    - **Tier 0:** 單機工具價值, 獲取用戶基數
    - **Tier 1:** 雲端服務價值, 變現安全與便利
    - **Tier 2:** 管理顧問價值, 變現預知能力與複雜性管理
    - **B2B:** 市場洞察價值, 變現總體經濟數據

## C 端商業模式：分層訂閱

- **Tier 0 免費版, Free Tier**
    - **核心哲學:** 獲取 Acquisition, 留存 Retention
    - **價值主張:** 提供市場上最快、無摩擦的單機記帳體驗，不以閹割核心記帳功能作為手段，而是以「工具的極致」作為護城河
    - **目標客群:** 價格敏感型用戶、輕量級使用者、重視隱私的單機派
    - **功能邊界:**
        - **React Native App:** 全功能開放
        - **資料庫:** 本地優先 WatermelonDB, 離線運作
        - **核心功能:** 極速記帳、基礎儀表板、Widget 小工具
        - **限制:** 無雲端同步、僅限單一裝置、僅限基礎 CSV 匯出
- **Tier 1 標準訂閱, Standard Premium**
    - **核心哲學:** 安全 Security, 便利 Convenience
    - **價值主張:** 買保險的概念。為核心用戶提供資料安全的避風港與跨裝置的無縫體驗
    - **目標客群:** 換機頻繁者、多裝置使用者 iPad + iPhone、資料安全焦慮者
    - **功能邊界:**
        - **包含 Tier 0 所有功能**
        - **雲端引擎:** 解鎖雲端同步 Sync Protocol
        - **備份:** 每日自動化雲端備照
        - **裝置:** 支援多裝置雙向同步
- **Tier 2 專業訂閱, Pro Premium**
    - **核心哲學:** 駕馭複雜性 Complexity, 預見 Foresight
    - **價值主張:** 個人企業 ERP。從單純的「紀錄者」升級為「經營者」，提供超越手機 App 限制的桌面級管理能力與 AI 預測
    - **目標客群:** 家庭財務管理者、投資組合複雜者、自由工作者、高資產族群
    - **功能邊界:**
        - **包含 Tier 1 所有功能**
        - **Web 控制台:** 解鎖複雜性管理平台
        - **進階工具:** JQL 查詢介面、自訂報表定義器
        - **AI 賦能:** 解鎖智慧運算與預測引擎
        - **預測能力:** 現金流時間序列預測、市場對標 Peer Comparison

## B 端商業模式：數據生態

- **企業數據服務, Enterprise Data Service**
    - **核心哲學:** 數據變現 Data Monetization
    - **價值主張:** 提供去識別化的真實消費場景數據，修正傳統問卷調查與滯後經濟指標的誤差
    - **目標客群:** 金融機構、市場研究公司、零售品牌商、學術單位
    - **收費模式:**
        - **訂閱制:** 定期存取市場情資儀表板
        - **流量制:** API 呼叫次數計費
    - **功能邊界:**
        - **B 端數據生態系:** 僅存取聚合後的統計數據
        - **市場情資儀表板:** 查詢特定受眾消費趨勢
        - **總經數據 API:** 串接即時通膨與資產流動指數

## 飛輪效應與互惠循環

- **數據飛輪:**
    - 更多 Tier 0 用戶加入 -> 累積更多微觀記帳數據 -> B 端數據產品精準度提升 -> B 端營收增加
- **研發回饋:**
    - B 端營收增加 -> 補貼 C 端研發與伺服器成本 -> C 端維持高品質免費版 -> 吸引更多 Tier 0 用戶加入
- **隱私承諾:**
    - 嚴格遵守 `no1_product_philosophy.md` 中的隱私優先原則，B 端獲利的前提是 C 端數據的絕對去識別化與安全，此為品牌信任的基石

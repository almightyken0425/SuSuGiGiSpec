# 搜尋: SearchScreen

## 畫面目標

- 提供搜尋所有交易與轉帳備註的介面，並作為導航至紀錄編輯畫面的入口

---

## 線框圖

```text
+--------------------------------+
| < Back   [ Search Note... ]    |
+--------------------------------+
| [Icon] Category Name           |
|        Note Text (Highlight)   |
|        $ Amount      Date      |
|                                |
| [Icon] Transfer                |
|        Note Text (Highlight)   |
|        $ Amount      Date      |
+--------------------------------+
```

---

## 佈局

### 導覽列

- 返回按鈕
- 搜尋輸入框
  - 進入時自動聚焦

### 搜尋結果區

- **IF** 搜尋框為空:
  - 顯示輸入關鍵字以搜尋交易備註提示
- **IF** 無符合結果:
  - 顯示找不到符合關鍵字的紀錄提示
- 搜尋結果列表項目
  - 圖示
  - 類別名稱或轉帳標籤
  - 備註
    - 關鍵字高亮顯示
  - 金額
  - 日期

---

## 互動

- **點按返回按鈕:**
  - 返回上一頁

- **輸入搜尋文字:**
  - 即時篩選並更新搜尋結果列表

- **點按搜尋結果列表項目:**
  - **IF** 交易紀錄:
    - 導航至 TransactionEditorScreen
  - **IF** 轉帳紀錄:
    - 導航至 TransferEditorScreen

# 帳戶列表規格: AccountListScreen

## 畫面目標

- **提供:** 查看, 排序, 管理 所有帳戶的介面
- **顯示:** 各帳戶的目前餘額
- **作為:** 新增, 編輯, 刪除 特定帳戶的入口

```text
+--------------------------------+
| < Back      Accounts           |
+--------------------------------+
| My Accounts        [+ Add]     |
| [Icon] Account Name      [=]   |
|        Type  $ Balance         |
| [Icon] Account Name      [=]   |
|        Type  $ Balance         |
|                                |
|                                |
+--------------------------------+
```

## UI 佈局

- **頂部導航列:**
    - **內部元件:**
        - **返回按鈕**
            - **導航:** SettingsScreen
        - **標題**
            - **內容:** 帳戶管理
- **頁面主體:**
    - **帳戶分區標題:**
        - **內部元件:**
            - **標題**
                - **內容:** 我的帳戶
            - **新增按鈕**
                - **觸發:** 付費牆檢查
                - **導航:** AccountEditorScreen
                - **模式:** 新增
    - **帳戶列表:**
        - **UI:** DraggableFlatList
        - **排序:** 依 `sortOrder` 排序
        - **項目內容:**
            - **內部元件:**
                - **圖示**
                - **名稱**
                - **帳戶類型**
                - **帳戶餘額**
                    - **備註:** 需格式化並顯示幣別
                - **拖拉圖示**
            - **視覺標示 - 停用:**
                - **條件:** `disabledOn` 非 null
                - **UI:** 灰色文字 或 已停用 標籤
        - **空狀態:**
            - **條件:** 沒有任何帳戶
            - **顯示:** 您尚未建立任何帳戶

## 核心邏輯

- **資料載入:**
    - **觸發:** 畫面載入
    - **來源:** 本機 DB
    - **查詢:** 所有 `Accounts`
    - **過濾:** `deletedOn` 為 null
    - **排序:** 依 `sortOrder` 初始排序
    - **計算:** 顯示每個帳戶的當前餘額
- **互動邏輯:**
    - **點擊項目:**
        - **導航:** AccountEditorScreen
        - **模式:** 編輯
        - **參數:** accountId
    - **拖拉排序:**
        - **行為:** 更新受影響帳戶的 `sortOrder`
        - **儲存:** 寫入 本機 DB
        - **欄位:** 必須更新所有受影響帳戶的 `updatedOn`
    - **點擊新增:**
        - **導航:** AccountEditorScreen
        - **模式:** 新增
- **付費牆檢查:**
    - **觸發:** 點擊 新增按鈕
    - **檢查:**
        - `PremiumContext.isPremiumUser` 狀態
        - 本機 DB 帳戶總數
    - **IF** 免費版 AND 帳戶数量已達上限:
        - **導航:** PaywallScreen
    - **ELSE:**
        - **導航:** AccountEditorScreen

## 導航

    - **來源:** SettingsScreen

    - **來源:** 點擊列表項目 或 新增按鈕
    - **導航:** AccountEditorScreen

        - **來源:** 點擊 新增按鈕

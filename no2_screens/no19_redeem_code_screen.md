# 序號兌換: RedeemCodeScreen

## 畫面目標

- 提供輸入序號並兌換會員權益的介面

---

## 線框圖

```text
+--------------------------------+
| < Back      Redeem Code        |
+--------------------------------+
|                                |
| [ XXXX-XXXX-XXXX             ] |
|                                |
| Enter 12-digit code from card  |
|                                |
|      [ Redeem Now ]            |
|                                |
| Recent History                 |
| - 2024/01/01 Tier 1            |
+--------------------------------+
```

---

## 佈局

### 導覽列

- 返回按鈕
- 兌換序號 標題

### 輸入區

- 序號輸入框
  - 自動格式化為 XXXX-XXXX-XXXX
- 序號位於實體卡片背面或活動信件中 說明文字

### 操作區

- 立即兌換 按鈕
  - **IF** 序號未填妥:
    - 不可點按
  - **IF** 操作中:
    - 顯示載入狀態

### 歷史記錄區

- 最近兌換記錄 標題
- 兌換記錄列表
  - 兌換日期與兌換項目

---

## 互動

- **點按返回按鈕:**
  - 返回上一頁

- **點按立即兌換按鈕:**
  - 呼叫 redeemCode
  - **IF** 操作中:
    - 顯示載入狀態
  - **IF** 操作成功:
    - 顯示兌換成功對話框
    - 關閉對話框後返回上一頁
  - **IF** 操作失敗:
    - 顯示序號無效或已過期提示

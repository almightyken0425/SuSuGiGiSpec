- 即時呈現財務總覽與收支紀錄，並提供快速新增交易與轉帳的入口

## 佈局

### Header

- 報表篩選按鈕
- Logo
- 搜尋按鈕
- 設定入口按鈕

### 主內容區

- 資料由 buildPeriodReport 產出
- 時間區間標題
- **環形圖區塊:**
  - 環形圖
  - 餘額數值，顯示於環形圖中心

### 逐筆紀錄

- 資料由 buildPeriodReport 產出
- **類型標題列:**
  - 類型名稱，支出或收入
  - 該類型小計金額
- **分組標題列:**
  - 各分組預設為收合狀態
  - 收合/展開圖示
  - **IF** 類別分組:
    - **IF** 一般類別:
      - 類別圖示
      - 類別名稱
    - **IF** 轉帳分組:
      - 外部帳戶圖示
      - 外部帳戶名稱
  - **IF** 日期分組:
    - 日期文字
  - 該分組小計金額
- **單筆紀錄列:**
  - **IF** 類別分組:
    - 金額
    - 備註
    - 日期
  - **IF** 日期分組:
    - **IF** 一般交易:
      - 類別圖示與類別名稱
      - 備註
      - 金額
    - **IF** 轉帳紀錄:
      - 方向圖示與轉出/轉入文字及外部帳戶名稱
      - 備註
      - 金額
- **IF** 無紀錄:
  - 顯示空狀態提示文字

### Footer

- **IF** 未啟用 Undo Mode:
  - 新增支出按鈕
  - 新增轉帳按鈕
  - 新增收入按鈕
- **IF** 啟用 Undo Mode:
  - 倒數計時圓圈
  - Undo 訊息文字
  - 關閉按鈕

### 載入狀態

- **IF** 初始載入中:
  - 顯示載入文字

## 互動

- **點按報表篩選按鈕:**
  - 導航至 HomeFilterScreen

- **點按搜尋按鈕:**
  - 導航至 SearchScreen

- **點按設定入口按鈕:**
  - 導航至 SettingsScreen

- **點按分組標題列:**
  - 展開或收合該分組
  - 類型標題列不可收合

- **點按單筆紀錄列:**
  - **IF** 一般交易:
    - 帶入紀錄資訊
    - 導航至 TransactionEditorScreen
  - **IF** 轉帳紀錄:
    - 帶入紀錄資訊
    - 導航至 TransferEditorScreen

- **點按新增支出按鈕:**
  - **IF** 為 LEVEL_0 且已超出免費使用限制:
    - 導航至 PaywallScreen
  - **ELSE:**
    - 帶入當前報表日期
    - 導航至 TransactionEditorScreen，支出模式

- **點按新增收入按鈕:**
  - **IF** 為 LEVEL_0 且已超出免費使用限制:
    - 導航至 PaywallScreen
  - **ELSE:**
    - 帶入當前報表日期
    - 導航至 TransactionEditorScreen，收入模式

- **點按新增轉帳按鈕:**
  - **IF** 為 LEVEL_0 且已超出免費使用限制:
    - 導航至 PaywallScreen
  - **ELSE:**
    - 帶入當前報表日期
    - 導航至 TransferEditorScreen

- **水平滑動分頁:**
  - 主內容區支援水平分頁滑動
  - 每頁代表一個時間區間
  - **IF** 時間粒度為全部:
    - 僅顯示一頁，不可滑動
  - 禁止向未來時間區間滑動
  - **IF** 無任何交易或轉帳紀錄:
    - 僅顯示當前時間區間，禁止向過去滑動
  - **IF** 有交易或轉帳紀錄:
    - 可向過去滑動至最早紀錄所在的時間區間
    - 向過去方向滑動時自動載入更多時間區間
  - 時間區間標題跟隨水平滑動

- **向上捲動主內容區:**
  - 環形圖區塊隨主內容區捲動消失

- **捲動回主內容區起點:**
  - 環形圖區塊顯示

- **點按 Undo 訊息文字:**
  - 呼叫 executeUndo

- **點按倒數計時圓圈或關閉按鈕:**
  - 呼叫 closeUndo

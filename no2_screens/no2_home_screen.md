## 佈局

### Header

- 報表篩選按鈕
- Logo
- 搜尋按鈕
- 設定入口按鈕

### 主內容區

- 時間區間標題
- **圓餅圖區塊:**
  - 圓餅圖
    - **IF** 圖表來源前五大類別:
      - 顯示彩色圖表色
    - **ELSE:**
      - 顯示預設色
  - 支出數值
  - 收入數值
- **總餘額區塊:**
  - 總餘額

### 逐筆紀錄

- 支出區與收入區的排列順序依圖表來源設定
- **分組標題列:**
  - 收合圖示
  - **IF** 類別分組:
    - 類別圖示
  - **IF** 日期分組:
    - 日期區間
  - **IF** 單幣別:
    - 原始幣別金額
  - **IF** 多幣別:
    - 主要貨幣換算金額
- **單筆紀錄列:**
  - **IF** 定期交易:
    - 循環圖示
    - **ELSE:**
      - 一般交易類型圖示
  - **IF** 類別分組:
    - **IF** 單幣別:
      - 原始幣別金額
    - **IF** 多幣別:
      - 原始幣別金額與主要貨幣換算金額
    - 備註
    - 日期
  - **IF** 日期分組:
    - 類別名稱
    - 備註
    - **IF** 單幣別:
      - 原始幣別金額
    - **IF** 多幣別:
      - 原始幣別金額與主要貨幣換算金額

### Footer

- 新增收入按鈕
- 新增轉帳按鈕
- 新增支出按鈕

## 互動

- **點按報表篩選按鈕:**
  - 導航至 HomeFilterScreen

- **點按分組:**
  - 展開或收合該分組

- **點按新增收入/支出/轉帳/紀錄按鈕:**
  - **IF** = Tier0 且超出免費使用限制:
    - 導航至 PaywallScreen
  - **IF** > Tier0:
    - **IF** 收支:
      - **IF** 新增紀錄:
        - 帶入當下時間
        - 導航至 TransactionEditorScreen
      - **IF** 既有紀錄:
        - 帶入紀錄資訊
        - 導航至 TransactionEditorScreen
    - **IF** 轉帳:
      - **IF** 新增紀錄:
        - 帶入當前報表日期
        - 導航至 TransferEditorScreen
      - **IF** 既有紀錄:
        - 帶入紀錄資訊
        - 導航至 TransferEditorScreen

- **滑動時間區間:**
  - 主內容區支援水平分頁滑動
  - 每頁代表一個時間區間
  - 禁止向未來時間區間滑動
  - **IF** 無任何交易或轉帳紀錄:
    - 僅顯示當前時間區間，禁止向過去滑動
  - **IF** 有交易或轉帳紀錄:
    - 可向過去滑動至最早紀錄所在的時間區間
  - 時間區間標題跟隨水平滑動

- **向上滾動主內容區:**
  - 圓餅圖區塊收合
  - Header 隱藏
  - Footer 隱藏
  - 逐筆紀錄展開填滿

- **滾動至主內容區頂部:**
  - 圓餅圖區塊展開
  - Header 顯示
  - Footer 顯示

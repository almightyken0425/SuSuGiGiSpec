# 類別編輯器: CategoryEditorScreen

## 畫面目標

- 提供新增或編輯類別的統一介面

---

## 線框圖

```text
+--------------------------------+
| Cancel      Title       Done   |
+--------------------------------+
| Name: [ Input Text ]           |
| Map To: [ Standard Cat ] v     |
| Icon: [ Icon Preview ] v       |
|                                |
| Disable: [ Switch ]            |
|                                |
|           [Delete]             |
+--------------------------------+
```

---

## 佈局

### 導覽列

- 取消按鈕
- 標題
  - **IF** 新增模式:
    - 新增支出類別或新增收入類別
  - **IF** 編輯模式:
    - 編輯支出類別或編輯收入類別
- 完成按鈕
  - **IF** 必填欄位未填妥:
    - 不可點按

### 欄位區

- 類別名稱輸入框
- 標準類別映射選擇器
  - **IF** 收合:
    - 顯示當前所選標準類別名稱與展開指示器
  - **IF** 展開:
    - 搜尋輸入框
    - 標準類別列表
- 圖示選擇器
  - **IF** 收合:
    - 顯示當前所選圖示與展開指示器
  - **IF** 展開:
    - 圖示網格

### 操作區

- **IF** 編輯模式:
  - 停用此類別 開關
  - 刪除此類別 按鈕

---

## 互動

- **點按取消按鈕:**
  - 返回上一頁

- **點按標準類別映射選擇器:**
  - 展開或收合標準類別列表
  - **IF** 展開:
    - 搜尋輸入框自動聚焦

- **輸入標準類別搜尋文字:**
  - 即時篩選標準類別列表

- **選擇標準類別列表項目:**
  - 收合選擇器並更新顯示

- **點按圖示選擇器:**
  - 展開或收合圖示網格

- **選擇圖示:**
  - 收合選擇器並更新預覽

- **點按完成按鈕:**
  - **IF** 新增模式:
    - 呼叫 canUserPerformAction，動作識別碼 createCategory
    - **IF** 回傳禁止:
      - 導航至 PaywallScreen
    - **ELSE:**
      - 呼叫 createCategory
      - **IF** 操作成功:
        - 返回上一頁
      - **IF** 操作失敗:
        - 顯示錯誤提示
  - **IF** 編輯模式:
    - 呼叫 updateCategory
    - **IF** 操作成功:
      - 返回上一頁
    - **IF** 操作失敗:
      - 顯示錯誤提示

- **點按刪除按鈕:**
  - 顯示刪除確認對話框
  - **IF** 確認刪除:
    - 呼叫 deleteCategory
    - **IF** 操作成功:
      - 呼叫 UndoLogic.showUndo
      - 返回上一頁
    - **IF** 操作失敗:
      - 顯示錯誤提示

# 帳戶編輯器: AccountEditorScreen

## 畫面目標

- 提供新增或編輯帳戶的統一介面

---

## 線框圖

```text
+--------------------------------+
| Close       Title       Save   |
+--------------------------------+
| Name: [ Input Text ]           |
| Currency: [ TWD ] v            |
| Type: [ Standard Type ] v      |
| Icon: [ Icon Preview ] v       |
|                                |
| Disable: [ Switch ]            |
|                                |
|           [Delete]             |
+--------------------------------+
```

---

## 佈局

### 導覽列

- 關閉按鈕
- 標題
  - **IF** 新增模式:
    - 新增帳戶
  - **IF** 編輯模式:
    - 編輯帳戶
- 儲存按鈕
  - **IF** 必填欄位未填妥:
    - 不可點按

### 欄位區

- 帳戶名稱輸入框
- 幣別選擇器
  - **IF** 收合:
    - 顯示當前所選幣別代碼與展開指示器
  - **IF** 展開:
    - 搜尋輸入框
    - 幣別列表
  - **IF** 編輯模式:
    - 不可修改
- 標準帳戶類型選擇器
  - **IF** 收合:
    - 顯示當前所選類型名稱與展開指示器
  - **IF** 展開:
    - 類型列表
- 圖示選擇器
  - **IF** 收合:
    - 顯示當前所選圖示與展開指示器
  - **IF** 展開:
    - 圖示網格

### 操作區

- **IF** 編輯模式:
  - 停用此帳戶 開關
  - 刪除此帳戶 按鈕

---

## 互動

- **點按關閉按鈕:**
  - 返回上一頁

- **點按幣別選擇器:**
  - 展開或收合幣別列表
  - **IF** 展開:
    - 搜尋輸入框自動聚焦

- **輸入幣別搜尋文字:**
  - 即時篩選幣別列表

- **選擇幣別列表項目:**
  - 收合選擇器並更新顯示

- **點按標準帳戶類型選擇器:**
  - 展開或收合類型列表

- **選擇類型列表項目:**
  - 收合選擇器並更新顯示

- **點按圖示選擇器:**
  - 展開或收合圖示網格

- **選擇圖示:**
  - 收合選擇器並更新預覽

- **點按儲存按鈕:**
  - **IF** 新增模式:
    - 呼叫 canUserPerformAction，動作識別碼 createAccount
    - **IF** 回傳禁止:
      - 導航至 PaywallScreen
    - **IF** 選擇外幣:
      - 呼叫 canUserPerformAction，動作識別碼 useForeignCurrency
      - **IF** 回傳禁止:
        - 導航至 PaywallScreen
    - 呼叫 createAccount
    - **IF** 操作成功:
      - 返回上一頁
    - **IF** 操作失敗:
      - 顯示錯誤提示
  - **IF** 編輯模式:
    - 呼叫 updateAccount
    - **IF** 操作成功:
      - 返回上一頁
    - **IF** 操作失敗:
      - 顯示錯誤提示

- **點按刪除按鈕:**
  - 顯示刪除確認對話框
  - **IF** 確認刪除:
    - 呼叫 deleteAccount
    - **IF** 操作成功:
      - 呼叫 UndoLogic.showUndo
      - 返回上一頁
    - **IF** 操作失敗:
      - 顯示錯誤提示

# 合併編輯器: MergeEditorScreen

## 畫面目標

- 提供合併兩個帳戶或兩個類別的介面

---

## 線框圖

```text
+--------------------------------+
| Cancel      Merge       Done   |
+--------------------------------+
|                                |
|  [ Source ] ->  [ Target ]     |
|                                |
|  Select Source  Select Target  |
|  [ Selector ]   [ Selector ]   |
|                                |
|  Warning:                      |
|  Merging will move all records |
|  from Source to Target.        |
|  Source will be deleted.       |
|                                |
+--------------------------------+
```

---

## 佈局

### 導覽列

- 取消按鈕
- 合併 標題
- 完成按鈕
  - **IF** 來源或目標未選擇:
    - 不可點按
  - **IF** 來源與目標相同:
    - 不可點按

### 合併操作區

- 來源項目圖示與名稱
- 箭頭
- 目標項目圖示與名稱
- 來源選擇器
  - 將被刪除 標籤
- 目標選擇器
  - 將保留 標籤

### 警告訊息區

- 合併後，所有屬於來源的交易紀錄將轉移至目標，且來源將被刪除

---

## 互動

- **點按取消按鈕:**
  - 返回上一頁

- **點按來源選擇器:**
  - 彈出項目選擇清單
  - 不顯示已停用項目
  - 不顯示已選為目標的項目

- **點按目標選擇器:**
  - 彈出項目選擇清單
  - 不顯示已停用項目
  - 不顯示已選為來源的項目

- **點按完成按鈕:**
  - **IF** 帳戶模式:
    - 呼叫 mergeAccounts
  - **IF** 類別模式:
    - 呼叫 mergeCategories
  - **IF** 操作成功:
    - 返回上一頁
  - **IF** 操作失敗:
    - 顯示錯誤提示

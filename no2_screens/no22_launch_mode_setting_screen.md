# 啟動模式設定: LaunchModeSettingScreen

## 畫面目標

- 提供選擇 App 啟動時預設顯示記帳模式的介面

---

## 線框圖

```text
+--------------------------------+
| X      Launch Mode       Done  |
+--------------------------------+
| Default                     ✓  |
| Expense                        |
| Income                         |
| Transfer                       |
+--------------------------------+
```

---

## 佈局

### 導覽列

- 關閉 按鈕
- 啟動模式 標題
- 完成 按鈕

### 啟動模式列表

- 啟動模式選項
  - 模式名稱：Default、Expense、Income、Transfer
  - **IF** 為目前選取模式:
    - 顯示選取標記

---

## 互動

- **點按關閉按鈕:**
  - 返回上一頁

- **點按啟動模式選項:**
  - 將此選項標為目前選取模式

- **點按完成按鈕:**
  - 呼叫 setLaunchMode
  - 返回上一頁

# 股權分配總表

## 核心目標

- 本文件是股權計算的最終總結
- 目的在於將 `no2_scope_value_ledger.md` 的任務分配給 `no3_role_definitions.md` 的角色
- 藉此計算出每個角色在總理論價值中所佔的價值點數比例
- 這個比例是未來新資分配與股權結構的依據

## 角色價值分配

- **說明:** 團隊必須共同討論, 將 `no2` 檔案中的總價值點數, 分配給 `no3` 中的各個角色。
- **範例:** 假設總理論價值為 1,000,000 點

| 角色 | 負責範疇 | 估算價值點數 | 佔總價值比例 |
| --- | --- | --- | --- |
| **領導與策略** | 領導、融資、法務 | 100,000 | 10.0% |
| **產品管理** | 產品規格 SuSuGiGiSpec | 120,000 | 12.0% |
| **專案管理** | 時程控管、跨部門協調 | 80,000 | 8.0% |
| **UI/UX 設計** | App 視覺、Web 視覺 | 100,000 | 10.0% |
| **前端開發** | React Native App | 200,000 | 20.0% |
| **後端開發** | 雲端後端、Web 平台 | 200,000 | 20.0% |
| **品質保證** | App 與 Web 測試 | 80,000 | 8.0% |
| **行銷與成長** | 使用者獲取、社群經營 | 100,000 | 10.0% |
| **財務與法務** | 公司設立、金流管理 | 20,000 | 2.0% |

# MVP 範圍定義

_(本文件定義 V1 上線必須包含的最小功能集，已根據「本地優先 + 混合付費牆」策略更新)_

## 核心架構 (In Scope for MVP)

- [ ] **本機儲存架構 (Local-First):** App 核心功能（CRUD）必須能在 100% 離線狀態下於「本機資料庫 (Local DB)」運作。
    
- [ ] **使用者認證:** 強制 Google 登入 (Firebase Auth)，用於帳號綁定。
    
- [ ] **付費牆 (PaywallScreen):** 串接 App Store / Google Play 訂閱機制 (例如 RevenueCat)。
    

## 免費版功能 (Free Tier)

- [ ] **核心記帳:**
    
    - [ ] 新增/編輯/刪除 **支出 (Expense)** 交易。
        
    - [ ] 新增/編輯/刪除 **收入 (Income)** 交易。
        
    - [ ] 新增/編輯/刪除 **同幣別** 轉帳 (Transfer)。
        
- [ ] **功能限制 (免費版):**
    
    - [ ] **帳戶限制:** 最多 3 個帳戶。
        
    - [ ] **類別限制:** 最多 10 個自訂類別。
        
- [ ] **首頁 (HomeScreen):**
    
    - [ ] 完整儀表板功能 (時間區間切換、帳戶篩選器、摘要/列表視圖)。
        
- [ ] **帳戶/類別管理:**
    
    - [ ] 建立/查看/編輯/刪除帳戶 (受限於 3 個)。
        
    - [ ] 建立/查看/編輯/刪除類別 (受限於 10 個)。
        
- [ ] **首次啟動流程:**
    
    - [ ] 偵測新用戶，並在**本機資料庫**建立預設幣別、時區、預設帳戶與類別。
        
- [ ] **設定 (MVP):**
    
    - [ ] 帳戶管理入口。
        
    - [ ] 類別管理入口。
        
    - [ ] 偏好設定入口 (包含主要貨幣、時區、語系切換、登出功能)。
        
    - [ ] **國際化 (i18n):** App 必須支援至少兩種語言（例如 `zh-TW`, `en`），並能根據使用者偏好設定顯示。
        
- [ ] **搜尋功能:**
    
    - [ ] 支援搜尋 `Note` 欄位（僅限本機資料）。
        

## 付費版功能 (Premium Tier)

- [ ] **[付費功能] 雲端同步與備份:**
    
    - [ ] 實作「**每日批次同步**」引擎。
        
    - [ ] 實作「**立即同步 (Sync Now)**」手動按鈕（包含 5 分鐘冷卻機制）。
        
- [ ] **[付費功能] 解鎖限制:**
    
    - [ ] **無限**帳戶。
        
    - [ ] **無限**類別。
        
- [ ] **[付費功能] 多幣別功能:**
    
    - [ ] 建立/管理外幣帳戶。
        
    - [ ] 支援**跨幣別**轉帳。
        
    - [ ] **匯率管理**介面。
        
- [ ] **[付費功能] 自動化:**
    
    - [ ] **定期交易 (`Schedules`)** 的建立、編輯/刪除及自動產生。
        
- [ ] **[付費功能] 資料遷移:**
    
    - [ ] **匯入資料 (`.csv` 匯入)** 功能。
        

## 延後功能 (Out of Scope for MVP)

- [x] **匯出資料** (`.csv` / `.db`)。此為付費功能，但延後至 Premium V1.1 釋出。
    
- [x] **快速捕捉小工具 (Widgets)**。
    
- [x] (未來) **AI 分析**。
    
- [x] (未來) **進階報表:** 獨立的「報表」頁面 (長條圖、趨勢圖等)。

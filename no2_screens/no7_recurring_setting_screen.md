# 定期設定區: RecurringOptions

## 畫面目標

- 提供嵌入於交易與轉帳編輯器中的定期規則設定區，控制此筆是否設為定期，以及定期的頻率、間隔與結束條件

---

## 觸發情境

- 嵌入於 TransactionEditorScreen 與 TransferEditorScreen，由所在畫面的定期觸發按鈕展開或收合

---

## 線框圖

```text
+--------------------------------+
| Recurring Settings    [Switch] |
+--------------------------------+
| Frequency                      |
| [Daily][Weekly][Monthly][Yearly]
|                                |
| Every                          |
| [  1  ] Month(s)               |
|                                |
| End                            |
| [ Never ] [ On Date ]          |
|                                |
| (Date Picker if On Date)       |
+--------------------------------+
```

---

## 佈局

### 標題列

- 區域標題
- 啟用開關

### 設定內容區

- **IF** 啟用開關為關:
  - 設定內容區半透明且不可操作
- **IF** 啟用開關為開:
  - 設定內容區可操作

- 頻率選項列
  - 每日選項
  - 每週選項
  - 每月選項
  - 每年選項

- 間隔列
  - 間隔輸入框
  - 單位文字
    - **IF** 頻率為每日:
      - 顯示 Day(s)
    - **IF** 頻率為每週:
      - 顯示 Week(s)
    - **IF** 頻率為每月:
      - 顯示 Month(s)
    - **IF** 頻率為每年:
      - 顯示 Year(s)

- 結束條件選項列
  - 永不選項
  - 於指定日期選項

- **IF** 結束條件為於指定日期:
  - 結束日期選擇器

---

## 互動

- **切換啟用開關為開:**
  - 呼叫 canUserPerformAction，動作識別碼 createRecurringTransaction
  - **IF** 回傳允許:
    - 啟用設定內容區並回傳目前規則至所在畫面
  - **IF** 回傳禁止:
    - 維持開關為關

- **切換啟用開關為關:**
  - 停用設定內容區並回傳清空狀態至所在畫面

- **點按頻率選項:**
  - 更新當前頻率為點按之選項

- **修改間隔輸入:**
  - 更新當前間隔為輸入值

- **點按永不選項:**
  - 清除結束日期

- **點按於指定日期選項:**
  - 預設結束日期為一年後

- **變更結束日期:**
  - 更新結束日期為新值

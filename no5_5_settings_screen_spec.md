# 設定主頁規格: SettingsScreen

## 畫面目標

- **提供:** 存取所有帳戶, 資料, 偏好設定的中心化入口
- **提供:** 登出與升級至付費版的路徑

## UI 佈局

- **頂部導航列:**
    - **內部元件:**
        - **返回按鈕**
            - **導航:** HomeScreen
        - **標題**
            - **內容:** 設定
- **設定列表:**
    - **UI:** 分組列表, SectionList
    - **內部元件 - 資料管理:**
        - **類別管理**
            - **UI:** 圖示 + 類別管理 文字
            - **導航:** CategoryListScreen
        - **帳戶管理**
            - **UI:** 圖示 + 帳戶管理 文字
            - **導航:** AccountListScreen
        - **匯率管理**
            - **UI:** 圖示 + 匯率管理 文字 + Premium 標籤
            - **導航:** CurrencyRateListScreen
            - **邏輯:** 點擊時檢查付費狀態
        - **匯入資料**
            - **UI:** 圖示 + 匯入資料 文字 + Premium 標籤
            - **導航:** ImportScreen
            - **邏輯:** 點擊時檢查付費狀態
    - **內部元件 - 偏好設定:**
        - **偏好設定**
            - **UI:** 圖示 + 偏好設定 文字
            - **導航:** PreferenceScreen
    - **內部元件 - 升級:**
        - **升級至 Premium**
            - **UI:** 醒目項目, 圖示 + 升級至 Premium 文字
            - **可見性:** 僅免費版使用者顯示
            - **導航:** PaywallScreen
    - **內部元件 - 帳號:**
        - **登出**
            - **UI:** 紅色 登出 文字按鈕
            - **觸發:** 登出邏輯

## 核心邏輯

- **付費功能檢查:**
    - **觸發:** 畫面載入
    - **行為:** 讀取 `PremiumContext.isPremiumUser` 狀態
    - **邏輯:**
        - **匯率管理 / 匯入資料:**
            - **IF** 免費版:
                - **導航:** PaywallScreen
        - **升級至 Premium:**
            - **行為:** 依 `isPremiumUser` 狀態動態顯示或隱藏
- **登出邏輯:**
    - **觸發:** 點擊 登出 按鈕
    - **行為:** 顯示確認對話框
    - **成功:**
        - **行為:** 呼叫 `authService.signOut()`
    - **監聽:**
        - **目標:** AuthContext 認證狀態變化
        - **行為:** AppNavigator 自動切換回 LoginScreen

## 導航

- **進入:**
    - **來源:** HomeScreen 設定圖示點擊
- **退出:**
    - **來源:** 頂部導航列 返回按鈕點擊
    - **導航:** HomeScreen
